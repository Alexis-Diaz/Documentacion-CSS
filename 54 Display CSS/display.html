Propiedad Display CSS

La propiedad display es la propiedad CSS más importante para controlar el diseño.

Esta propiedad especifica si se muestra o cómo se muestra el elemento HTML.

Cada elemento HTML tiene un valor de visualización predeterminado según el elemento que sea. El valor de visualización predeterminado para la mayoría de los elementos es block o inline.

Display:block

Un elemento a nivel de bloque siempre comienza en una nueva línea y ocupa todo el ancho disponible (se extiende hacia la izquierda y derecha tanto como puede.).

Display:none

La propiedad display: none; se usa comúnmente con JavaScript para ocultar y mostrar elementos sin eliminarlos y volver a crearlos. Ciertos elementos tiene un display con un valor "none" por ejemplo la etiqueta <script>.

Cambiar el display

La propiedad display se puede cambiar cuando queramos si importar el elemento. Esto puede ser útil para acomodar mejor la información en la pantalla. Un ejemplo común es cambiar el display:block por defecto en los elementos <li> de una lista y pasarlos a display:inline para crear barra de menús horizontales.

display:none o visibility:hidden

Display:none oculta el elemento y la página hace como si el elemento no existiera, por lo que no afecta al diseño una vez esta oculto con esta propiedad.

Visibility:hidden oculta el elemento pero aunque no se ve aun sigue ocupando el espacio, esto significa que aún afectará al diseño de la página.
