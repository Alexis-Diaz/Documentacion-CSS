Position

La propiedad position especifica el tipo de método de posicionamiento utilizado para un elemento (estatic, relative, fixed, absolute o sticky).

Luego, los elementos se colocan utilizando las propiedades superior, inferior, izquierda y derecha. Sin embargo, estas propiedades no funcionarán a menos que position se establezca primero. También funcionan de forma diferente según el valor de la posición.

position:static

Los elementos HTML se colocan estáticos de forma predeterminada. Los elementos de posición estática no se ven afectados por las propiedades superior, inferior, izquierda y derecha. Un elemento con position: static;no se posiciona de ninguna manera especial; siempre se posiciona de acuerdo con el flujo normal de la página:

Elemento div static

position:relative

Un elemento con position: relative;está posicionado en relación con su posición normal. Establecer las propiedades superior, derecha, inferior e izquierda de un elemento en una posición relativa hará que se ajuste lejos de su posición normal. El resto del contenido de la pagina no se ajustará para encajar en ningún espacio dejado por el elemento.

Elemento div relative

position:fixed

Un elemento con position: fixed;está posicionado en relación con la ventana gráfica, lo que significa que siempre permanece en el mismo lugar incluso si se desplaza la página. Las propiedades superior, derecha, inferior e izquierda se utilizan para colocar el elemento. Un elemento fijo no deja un hueco en la página donde normalmente se habría ubicado. Observe el elemento fijo en la esquina superior derecha de la página. Aquí está el CSS que se utiliza: 

            #divFixed{
                position: fixed;
                right: 0;
                top:0;
                border: solid 2px green;
                padding-top: 20px;
                padding-left: 10px;
                height: 60px;
                width: 70px;
                border-radius:40px;
                color: white;
                background-color: rgb(13, 70, 5);
            }

Elemento div fixed

position:absolute

Un elemento con position: absolute; está posicionado en relación con el antepasado posicionado más cercano (en lugar de posicionado en relación con la ventana gráfica, como fijo). Sin emabargo; si un elemento de posición absoluta no tiene antepasados ​​de posición, utiliza el cuerpo del documento y se mueve junto con el desplazamiento de la página. Nota: Un elemento "posicionado" es aquel cuya posición es cualquier cosa excepto static. A continuación, se muestra un ejemplo sencillo:

position:sticky (pagajosa)

Un elemento con position: sticky;se posiciona según la posición de desplazamiento del usuario. Un elemento sticky alterna entre relative y fixed, según la posición de desplazamiento. Se coloca en una posición relativa hasta que se alcanza una posición de desplazamiento determinada en la ventana gráfica, luego se "pega" en su lugar (como la posición: fija).

Elementos superpuestos

Cuando se colocan elementos, pueden superponerse a otros elementos cuando modificamos su position en relative, absolute, fixed o sticky. Nota: Lo anterior no aplica para position:static. Ellos siempre queda abajo de las demás posiciones. Observar por ejemplo el elemento div sticky que pasa por encima de los elementos que están después de él, pero no así del div que contiene la imagen de megaman ya que tienen position relative. La imagen pasa abajo porque recordemos que tiene un z-index: -1;.
Cuando agregamos elementos html estos se sobreponen respetando el orden en que se encuentran. Por ejemplo el elemento que se encuentra en la segunda posición se sobrepondrá al primero, pero se colocará abajo del tercero parecido a escamas del pescado.

La propiedad z-index especifica el orden de pila de un elemento (qué elemento debe colocarse delante o detrás de los demás). Un elemento puede tener un orden de pila positivo o negativo. En el siguiente ejemplo, el título se sobrepone a la etiqueta <img> porque a la imagen le asignamos la propiedad z-index con valor -1, lo que hace que se ubique por debajo del elemento div y el elemento h3. 

                div {
                    position: relative;
                    width: 500px;
                    height: 400px;
                    border: rgb(0, 45, 128) solid 2px;
                }
                
                h3 {
                    text-align: center;
                    font-size: 30px;
                    margin-top: 10px;
                    color: white;
                }

                img {
                    position: absolute;
                    top:10px;
                    left: 50px;
                    z-index: -1;
                    width: 400px;
                }
Notar que pareciera que el elemento div no afecta a la imagen, pero esto se debe a que el div es transparente. Si se le cambiara el color de fondo veremos que efecto se encuentra posicionado sobre la imagen.
Por tanto el orden de mas abajo hacia arriba de los siguientes elementos es el siguiente:
  1. Elemento img
  2. Elemento div
  3. Elemento h3
