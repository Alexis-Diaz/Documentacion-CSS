Propiedades del contenedor flexible

Un contenedor que tiene la propiedad display: flex cuenta con 6 propiedades diferentes. Cada una con un comportamiento especifico y util.

  1. flex-direction
  2. flex-wrap
  3. flex-flow (shorhand para las dos anteriores)
  4. justify-content
  5. align-item
  6. align-content
