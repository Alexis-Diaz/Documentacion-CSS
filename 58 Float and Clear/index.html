float and clear

La propiedad float CSS especifica cómo debe flotar un elemento. La propiedad clear CSS especifica qué elementos pueden flotar junto al elemento borrado y en qué lado.

float

La propiedad float se utiliza para posicionar y dar formato al contenido, por ejemplo, dejar que una imagen flote a la izquierda del texto en un contenedor. La propiedad float puede tener uno de los siguientes valores:

  • left: El elemento flota a la izquierda del contenedor.
  • right: El elemento flota a la derecha del contenedor.
  • none: El elemento no flota esto es predeterminado.
  • inherit: El elemento hereda el valor flotante de su padre.

En su uso más simple, la propiedad float se puede usar para ajustar texto alrededor de imágenes.
