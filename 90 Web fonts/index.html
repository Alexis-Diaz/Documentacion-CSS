fuentes web

Cuando se necesita usar una fuente que no se encuentre en el navegador del cliente se, o si se desea personalizar una fuente, se puede usar una fuente propia. Para hacerlo se usa la regla CSS @font-face esta regla hara que el navegador cliente descargue la fuente que necesita.

Uso

En la @font-face regla; primero defina un nombre para la fuente (por ejemplo, myFirstFont) y luego apunte al archivo de fuente. Consejo: utilice siempre letras minúsculas para la URL de la fuente. Las letras mayúsculas pueden dar resultados inesperados en IE. Para usar la fuente para un elemento HTML, consulte el nombre de la fuente (myFirstFont) a través de la font-family propiedad:

Por efecto de que no he descargado una fuente solo dejo el codigo css que se utiliza. En @font-face se indica la nueva fuente nombrandola myFirstFont. En * se utiliza la nueva fuente.

<style>

@font-face {
font-family: myFirstFont;
src: url(sansation_light.woff);
}

* {
font-family: myFirstFont;
}

</style>
