Outline (contorno)

Un contorno es una línea dibujada fuera del borde del elemento. Ejemplo:

Este elemento <div> tiene un borde color negro y un contorno (outline) verde.

Un contorno es una línea que se dibuja alrededor de los elementos, FUERA de los bordes, para hacer que el elemento destaque.

Css tiene las siguientes propiedades de outline:

  • outline-style
  • outline-color
  • outline-width
  • oultine-offset
  • outline

Importante:Outline difiere de border. A diferencia de border, el outline se dibuja fuera del border del elemento y puede superponerse a otro contenido. Además, el contorno NO es parte de las dimensiones del elemento . La altura y ancho total del elemento no se ven afectados por el ancho de outline.

outline-style

La propiead outline-style especifica el estilo del contorno del control y puede tener uno de los siguientes valores:

  • dotted: Define un contorno punteado.
  • dashed: Define un contorno de lineas cortadas.
  • solid: Define un contorno solido.
  • double: Define un contorno doble.
  • groove: Define un contorno acanalado 3D.
  • ridge: Define un contorno estriado en 3D.
  • inset: Define un contorno de insercción en 3D.
  • outset: Define un contorno inicial 3D.
  • none: Define ningun contorno.
  • hidden: Define un contorno oculto.
