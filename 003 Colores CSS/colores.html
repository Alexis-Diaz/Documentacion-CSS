Colores

Los colores en CSS se pueden especificar usando nombres ya definidos o usando valores.

Los nombres pueden ser dificil de aprender, por lo que sera mejor utilizar los valores de colores.
Los valores de colores se pueden representar de cinco formas:

  1. Valores RGB (red green blue)

    2. Los valores RGB se cuentan desde el 0 hasta el 255

    Sintaxi rgb(222 122 123)

  2. Valores HEX (sistema hexadecimal)

    3. El valor hexadecimal comienza con el valor mas bajo 00 hasta el valor mas alto ff. La forma de contar es la siguiente:
