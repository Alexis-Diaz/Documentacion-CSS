Trnasformaciones

La propiedad transform permite mover, rotar, escalar y sesgar elementos.

Se pueden utilizar los siguientes metodos.

  • translate()
  • rotate()
  • scaleX()
  • scaleY()
  • scale()
  • skewX()
  • skewY()
  • skew()
  • matrix()

translate()

Este metodo mueve el elementos desde su posicion inicial hasta los parametros dados para el eje x y eje y.

El siguiente ejemplo mueve el elemento div 50 px a la derecha y 20 px hacia abajo desde su posicion actual.

translate (50px, 20px)

rotate()

Este metodo hace girar el elemento en sentido horario o antihorario segun grados dados.

El siguiente ejemplo rota el div en el sentido de las agujas del reloj 20 grados.

rotate(20deg)

El siguiente ejemplo rota el div en el sentido contrario de las agujas del reloj 20 grados.

rotate(20deg)

scale()

Este metodo mueve aumenta o disminuye el tamaño de un elemento (según los parametros dados para el ancho y el alto)

El siguiente ejemplo aumenta el elemento div a dos veces su ancho original y tres veces su altura.

scale (2, 3)

Existen dos metodos derivados de este: scaleX, scaleY. Estos metodos funcionan igual que el anterior, la diferencia es que cada uno solo modifica una dimension ancho (scaleX) y su alto (scaleY).

skewX y skewY

Estos metodos sesgan los elementos a lo largo de su eje X o eje Y. Ejemplo:

skewX (20deg)
skeY (20deg)

skew()

Este metodo es la combinaciones de los dos metodos anteriores.

ske (20deg, 10deg)

matrix()

Este metodo combina todos los metodos de transformacion 2D en uno. Toma por tanto, seis parametros que contienen funciones matematicas, lo qe le permite rotar, escalar, mover, y sesgar elementos.

Los parámetros son los siguientes: matrix (scaleX (), skewY (), skewX (), scaleY (), translateX (), translateY ())

ske (20deg, 10deg)
