Font Style

La propiedad font-style se usa principalmente para especificar text en cursiva.

Esta propiedad tiene tres valores:

  • normal (normal): texto mostrado normalmente.
  • italic (cursiva): text mostrado en cursiva
  • oblique (oblicuo): texto es inclinado (similar a la cursiva, pero menos compatible)
