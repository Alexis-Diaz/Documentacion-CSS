Text Shadown

La propiedad text-shadown agrega sombra al texto.

En su uso más simple, solo especifica la sombra horizontal (2px) y la sombra vertical (2px). Se pueden usar valores negativos.

Ejemplo:

text-shadow: 2px 2px;


Esta propiedad tambien admite colores para la sombra. Notar que se usa un valor negativo para la posición arriba-abajo.

text-shadow: 7px 9px rgb(200,200,200);

text-shadow: 7px -9px rgb(200,200,200);


Tambien se le puede agregar desenfoque. Aqui se usa desenfoque de 5px. Ejemplo:

text-shadow: 2px 2px 5px red
