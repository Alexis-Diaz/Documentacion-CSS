text-overflow

La text-overflow propiedad CSS especifica cómo se debe señalar al usuario el contenido que no se muestra.

Contenido recortado usando el valor clip.
Es valor es por defecto, es decir que cuando se usa la overflow: hidden sobre el parrafo, de manera automatica el texto se corta, esto sucede porque text-overflow por defecto es clip. Para comprobarlo basta con borra la propiedad de la hoja de estilo y se verá que no cambia nada.

word-wrap

Esta propiedad CSS permite que las palabras largas puedan dividirse y ajustarse a la siguiente línea.

Si una palabra es demasiada larga para caber dentro de un area, se expande hacia afuera.
