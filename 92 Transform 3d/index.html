Transformaciones 3D

Con la transformpropiedad CSS puede utilizar los siguientes métodos de transformación 3D:

  • rotateX()
  • rotateY()
  • rotateZ()

rotateX

Este metodo rota el elemento sobre su eje x

rotateX(150deg)

rotateY

Este metodo rota el elemento sobre su eje y

rotateY(150deg)

rotateZ

Este metodo rota el elemento sobre su eje z

rotateZ(90deg)
